SomewhatHuman

This is a quick and dirty adaptation of MakeHuman to the Web.

SomewhatHuman is a modified version of Makehuman prepared to be executed as a service.

I have rescued this project from my old hard drive. It is a proof of concept I prepared some time ago.

It is not intended for real world use, but as starting point or just for fun.

  • Only a small set of actions is available. All the QT user interface has been removed and only a few options have been implemented, but it would be possible to enable more options following the same pattern.
  • It is based in a old version of MakeHuman (2014)
  • Uses three.js for rendering
  • It is possible to run it locally, using winpython, for instance. Requires Flask, Numpy & Scipy
